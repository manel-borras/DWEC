URL
the address of a web page.
HTTP
The communications protocol used to connect to Web servers on the Internet or on a local network
TCP/IP
a set of rules that governs the connection of computer systems to the internet.
Internet
electronic communications network that connects computer networks and organizational computer facilities around the world
W3C
The World Wide Web Consortium (W3C) is an international community where Member organizations, a full-time staff, and the public work together to develop Web standards.
HTML
Hypertext Markup Language, a standardized system for tagging text files to achieve font, colour, graphic, and hyperlink effects on World Wide Web pages.
